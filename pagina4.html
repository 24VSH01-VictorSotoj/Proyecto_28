  • Dominio:

    En HTML, el término "dminio" se refiere al nombre de dominio web, que es la dirección de un sitio web en internet.

  • Sub dominio:

    Un subdominio es una nueva dirección o URL que reemplaza el "www" de tu dominio por la palabra que desees. Por ejemplo, "tienda" en lugar de "www".

  • URL

    Es la dirección de un recurso específico en la web, como una página web, una imagen o un archivo. Es la cadena de texto que se utiliza para acceder a ese recurso en Internet.

  • DNS

    "Domain Name System" (Sistema de Nombres de Dominio), es como la guía telefónica de Internet. Convierte los nombres de dominio fáciles de recordar, como "google.com", en las direcciones IP numéricas que las computadoras usan para comunicarse.

  • Hosting

    El hosting, o alojamiento web, es un servicio que te permite almacenar y publicar los archivos de tu sitio web en Internet.

  • FTP

    FTP, que significa File Transfer Protocol o Protocolo de Transferencia de Archivos, es un protocolo de red que permite la transferencia de archivos entre sistemas en una red, especialmente Internet.

  • HTTP

    HTTP, o Protocolo de Transferencia de Hipertexto, es un protocolo de comunicación utilizado para la transferencia de datos en la web.

  • HTTPS

    HTTPS significa "Hypertext Transfer Protocol Secure", o Protocolo Seguro de Transferencia de Hipertexto en español. Es la versión segura del protocolo HTTP, que se utiliza para la transferencia de información en la web.

  • W3C

    El W3C, o World Wide Web Consortium, es una organización internacional que desarrolla estándares web para la World Wide Web. Su objetivo es garantizar el crecimiento a largo plazo de la web, la interoperabilidad de las tecnologías web y la accesibilidad para todos los usuarios.

  • UTF-8

    UTF-8 es un estándar de codificación de caracteres que permite representar casi cualquier carácter de Unicode utilizando una secuencia de 1 a 4 bytes.

  • HTML 5 y Estructura

    HTML5 es la última versión del Lenguaje de Marcado de Hipertexto (HTML), que se utiliza para estructurar y presentar contenido en la web. Su estructura básica incluye un encabezado () y un cuerpo (). El encabezado contiene información sobre el documento HTML, como el título de la página, mientras que el cuerpo contiene todo el contenido visible en la página.
